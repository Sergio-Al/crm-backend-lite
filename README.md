## Description

API simulating a CRM with less complex structure

## Installation

```bash
$ npm install
```

## Requirements
- Docker
- PostgreSQL (through docker)

## Running the app
```bash
# database
$ docker compose up -d
```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
